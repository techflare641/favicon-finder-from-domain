# Favicon Finder Frontend

React/TypeScript frontend for the Favicon Finder application.

## Setup

```bash
npm install
npm run dev
```

## Building for Production

```bash
npm run build
npm run preview
```

## Configuration

Create a `.env` file to configure the backend API URL:

```
VITE_API_URL=http://localhost:3001
```

## Features

- CSV file upload (supports files without headers)
- Real-time processing status display
- Results statistics (found, not found, errors)
- Automatic CSV download with results
- Modern, responsive UI design
